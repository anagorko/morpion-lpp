Workers

Quick and dirty guide on how to run a worker on a tree.
  1. Clone branch gemmate from https://github.com/anagorko/morpion-lpp 
        $ git clone https://github.com/anagorko/morpion-lpp.git
    $ cd morpion-lpp
    $ git pull origin gemmate
  2. Build generator 
        $ mkdir build
    $ cd build
    $ cmake ../src
    $ make
  3. Install python requests package
  4. Install gurobi (gurobipy included) and a gurobi license
  5. Run worker 
        $ cd ../src/gemmate
    $ ./worker.py --tree "tree id" --host "hostname"
